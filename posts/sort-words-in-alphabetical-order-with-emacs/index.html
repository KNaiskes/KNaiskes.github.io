Sort Words in Alphabetical Order With Emacs

Posted on 23 February, 2021

A couple of days ago, I had a project with a list of hundreds of books titles and decided, instead of sorting them during the runtime, it would be better to be sorted inside the source code.

My very first thought was to use Python or Bash as I have used both of these scripting languages in the past for such tasks and while I was waiting for Emacs to load (just a couple of seconds) and was thinking of how I should implement this simple script. I instantly remembered that Emacs is well known for being a great Operating System. a Lisp interpreter and as result such functionality could have already been implemented to Emacs and actually, I was right.

How to

Select lines - Visual select

Control - Space

Sort lines

M-x sort-lines RET
